What does this script do?
This script makes directories in the parent directory for codes (e.g. university courses codes) provided in the codes.txt file and moves the files in the parent directory containing any code to its respective directory case insensitively.

How to setup codes.txt file?
Create codes.txt once in directory as the script coursesFileOrganizer.py in.
Write your codes having space between them or on new line.
Save codes.txt file.
codes.txt can be edited if you want to change codes.

How to run the script?
Make sure python is installed.
Open the command terminal and make sure to be in the same directory as coursesFileOrganizer.py in.
Command to run the script:
python coursesFileOrganizer.py "parent directory path"
e.g. python coursesFileOrganizer.py C:\Users\XYZ\Downloads
